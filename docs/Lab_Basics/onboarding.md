# Onboarding: New Coordinators
Welcome to the Baller Lab! Here, we will outline the onboarding steps required during your first few weeks in the lab. New coordinators will need access to many different resources, many of which are password protected. It is a good idea to get started on creating these accounts/ having these created for you as soon as possible after joining the lab.

***TIP**: Keep track of all your usernames and passwords - there will be many!*

### Overview:

1. Admin (log in and access)
2. WorkDay trainings
    1. CITI, HIPAA, etc.
3. Databases/serves introduction
    1. Saturn, Oracle, REDCap, DaySmart, etc.
4. Study/visit related training
    1. Recruitment
    2. Consent
    3. CNB
    4. CAT-GOASSESS
    5. Payment
  
### To-Do’s:

1. Obtain ID’s
- [ ]  HUP (employee entrance).
- [ ]  PennCard (2nd floor of Penn Bookstore).
- [ ]  Once you get this, make sure you have access to Richards. If not, email Stace Moore asking for access.*
- [ ]  Send your Penn ID number to the research coordinator.
   
2. Obtain key’s from Stace Moore.
3. Schedule Onboard@Penn appointment.
4. Attend section orientation (Sage will tell you when this is, this is detailed below).
5. Meet with systems admin for computer launch/startup/sign in:
- [ ]  VPN
- [ ]  Slack
- [ ]  Outlook
- [ ]  Saturn
- [ ]  Printers
- [ ]  Etc.
6. Meet with Dr. Baller and complete introductory review period (supervisor initiates this in WorkDay).
7. Update Penn Directory (see [here](https://medley.isc-seo.upenn.edu/directory/jsp/fast.do?fastStart=profile)) and UPenn Emergency Alert System (see [here](https://www.publicsafety.upenn.edu/pennready/upennalert/)).
8. Complete Penn Learner Group module ([WorkDay](https://workday.upenn.edu)).
9. Send headshot and brief bio to Stace Moore via email.
10. Review employee handbook.
11. Register for Penn SOM Orientation ([WorkDay](https://workday.upenn.edu)).
12. Review institutional policies (e.g., benefits) (see [here](https://www.hr.upenn.edu/PennHR/benefits-compensation))
